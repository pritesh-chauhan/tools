<h1>Problem Statement</h1>

<p>Create a simple Nodejs/Express REST API for a simple todo app.</p>

<h1>User Requirements:</h1>

<li>As a user, I should be able to see all todo items fetched using a REST API.</li>
<li>As a user, I should be able to click a todo item and able to see its detailed view.</li>
<li>As a user, I should be able to open add a new todo item view by clicking the add button.</li>
<li>As a user, I should be able to add a todo item by entering the title, description, due date and time.</li>
<li>As a user, I should be able to mark a todo item as complete.</li>

<h1>Technical Requirements:</h1>

<li>The goal of this assignment is to learn about Angular.</li>
<li>Use angular HTTP client module to fetch data from the Nodejs server.</li>
<li>You should use angular component and data binding features.</li>
<li>You should use SCSS for CSS.</li>
<li>You should use angular-cli for building the project.</li>
<li>No JavaScript libraries should be used for this assignment.</li>
